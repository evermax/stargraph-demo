Stargraph Project

View on GitHub

About

This project allows you to get the stars from a Github repository and plot a graph out of it.

Important! The project only get the current stars and place them on the graph depending on its timestamp. It does not take into account the stars that were removed. That is why the graph is always growing or constant but not going down.

You can try it by either providing a Github API token or use the button at the top to sign up with Github.
